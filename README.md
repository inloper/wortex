# wortex

Client side - vuejs
Server API - python flask

### Installation:
client - npm install
server(api) - pip install -r requirements.txt

### Usage(localy):
**Server(API):**
λ: .\wortex\server\venv\Scripts\activate (on windows machine)
λ: \wortex\server\python run.py

**Client:**
λ: \wortex\client\nmpm run serve
